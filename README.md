# PSY 543 Clinical Research Methods 2026

Files related to Rick Gilmore's presentation in PSY 543.

## Contents

- `*.qmd`: Quarto files
  - `index.qmd` is the main talk file
- `*.html`: Rendered talk files
- `_metadata.yml`: YAML parameter file
- `_extensions/`: Quarto extensions
- `_freeze/`: Quarto data for freezing executions
- `include/`: Supporting resources
  - `bib/`: BibTex reference files
  - `css/`: CSS files
  - `fig/`: Figures generated by the code
  - `img/`: External figures
  - `pdf/`: External PDF
  
## Rendering

- Clone the repository to a local project directory.
- Render the slides
  - From terminal: `quarto render`
- The rendered HTML file is in `index.html`

The rendered site using GitHub pages can be found here:

<https://psu-psychology.github.io/psy-543-clinical-research-methods-2026/>
